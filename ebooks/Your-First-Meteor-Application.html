Your First Meteor Application

A free, online book about the Meteor JavaScript framework that helps beginning web developers build their first real-time web application with Meteor in a matter of hours.

Inside, you'll find:

  1. 24,000+ words, minus the bloat that plagues most technical books.
  2. Training that's precisely written for beginning web developers.
  3. Regular updates as the Meteor framework continues to evolve.
  4. Practical examples, real-world code samples, and helpful exercises
